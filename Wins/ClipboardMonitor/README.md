# GlimpseMe

你在电脑上留下的每一个痕迹，都在告诉AI你是谁。

---

我们相信，文字内容最大的读者将是AI，而不是人。

人可能只读一次，甚至不会读完。但AI可以读无数次，可以记住，可以理解，可以连接。你今天标注的一句话，明天的AI会记得，会和你三个月前标注的另一句话联系起来，形成对你这个人的理解。

现在的问题是：AI不知道你。它不知道你在意什么，你怎么思考，你对什么有态度。你和它对话，但它不认识你。每次对话都是从零开始。

GlimpseMe 解决这个问题。它收集你在电脑上的思想碎片——你选中的内容、你写下的想法、你点的赞——形成一份持续积累的个人数据。这份数据是给AI读的。

---

具体怎么用：在任何应用里选中任何内容，按快捷键，写下想法。或者只是点个赞，表示你在意这个东西。系统自动记录上下文：这段话来自哪个网页、哪个聊天对象、哪个文档。

所有数据存成结构化的JSON，干净、可读、易于AI处理。一份数据源，不重复存储，可以有多种展示方式。

---

我们还不知道最终是哪家公司的AI会用到这份数据。可能是OpenAI，可能是Anthropic，可能是未来某个还不存在的产品。但我们相信这条路是对的：个人数据层，独立于任何一家AI公司，属于你自己，由你决定什么时候、给谁读。

技术上，我们对常用应用做专门优化，能获取更丰富的上下文。目前支持浏览器、微信、Notion、VSCode。其他应用也能工作，只是信息会少一些。这个优化过程很快，最多两个月覆盖主流应用。
